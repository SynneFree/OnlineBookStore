# B/S课程 - 课程设计

实现一个二手书籍交易网站

<br>

### 项目结构
参考[文档设计](https://github.com/SynneFree/OnlineBookStore/blob/master/Design.pdf)

##运行环境
- 操作系统	: MacOS Mojave
- Python版本 : Python3.7
- 前端开发框架 : Flask框架
- 数据库平台 : MySQL
- 数据库管理软件 : Navicat
- 开发工具 : Sublime Text
- 文档编写 : MacDown
- 浏览器 	: Chrome, Firefox 和 Safari
